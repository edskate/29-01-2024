A tag a cria uma ligação entre uma página e outra.
Quando o usuário ckica em um link ele é redirecionado para outra página.

A URL para onde a página será redirecionada que pode ser uma página web

target: Propriedades:

Por padrão ao clicar em um link a nova página é aberta na mesma aba do navegador , ou seja substituindo a página atual.
