Tag Title

A tag title é responsável por definir o titulo de uma página web.

O conteúdo do titulo de uma página é muito importante para a otimização do mecanismo de pesquisa (SEO)! O titulo da página é usado pelos algoritmos de pesquisa para decidir a ordem ao listar as páginas nos resultados da pesquisa.

A tag title:

é utilizada dentro de outra tag que nós chamamos de head.
head é uma tag que contém informações gerais sobre o documento.
O titulo da página (tag title) é uma dessas informações

Definindo um title/cabeçalho.

Os titulos ou cabeçalhos são utilizados para identificar o assunto de uma página ou seção.
